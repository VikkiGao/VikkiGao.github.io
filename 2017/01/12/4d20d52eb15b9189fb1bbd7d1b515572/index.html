变量Variable介绍：1-局部变量与全局变量

<h2>1、局部变量（Local Variable）</h2>

局部变量也称为内部变量，局部变量在方法内部声明，作用域仅仅限于方法内。有关局部变量有如下几个常用的要点：



  • 局部变量在方法内部定义，只有在方法运行时才存在；

  • 局部变量没有默认的初始值，因此在使用前需要赋值；换句话说，当每次调用该方法时，局部变量都会被声明且初始化一次；

  • 在一个方法中，方法中的输入参数也属于局部变量的范畴；


例如：定义一个MyClass类，在该类中定义一个printlocalVariable方法，具体实现代码如下：



  • .h文件


@interface MyClass : NSObject

-(void) printlocalVariable;

@end


  • .m文件：定义一个局部变量localVar，并赋初始值0。当方法被调用时，打印当前localVar的值，之后localVar值+1；


-(void)printlocalVariable{
    int localVar = 0;
    NSLog(@"localVar value :%d",localVar);
    localVar++;

}


  • main(）函数中【多次】调用printlocalVariable方法


#import <Foundation/Foundation.h>
#import "MyClass.h"

int main(int argc, const char  argv[]) {
    @autoreleasepool {
        MyClass myClass = [[MyClass alloc] init];

        [myClass printlocalVariable];
        [myClass printlocalVariable];
        [myClass printlocalVariable];

    }
    return 0;
}



  • 打印结果如下。每次调用方法时，localVar都会被重新初始化赋值，因此每次打印值都为0。



2、全局变量(Global Variable)


全局变量也成为外部变量，它不属于任何一个方法，而是属于一个源程序文件，其作用域是整个程序。


在iOS开发中，全局变量的实现主要有两种实现方式，第一种是使用单例模式实现（非常常用，详解其他课程），第二种是使用关键字extern。


当使用关键字extern定义全局变量时，变量名建议以小写字母g开头，如下所示：


extern NSString *gExternString;
